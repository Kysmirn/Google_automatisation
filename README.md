# Google Automation Portfolio

A collection of lightweight business automation solutions built with:
- Google Sheets
- Google Apps Script
- JavaScript
- clasp
- Git/GitHub
This repository contains production-inspired automation cases focused on improving operational workflows for small teams and businesses.
---
# Repository Structure
## `vacation-tracker`
Google Sheets based vacation management system with:
- automated Telegram notifications;
- overlap validation;
- duplicate prevention;
- backend-driven validation logic;
- Apps Script automation workflow.

### Main features
- Vacation overlap validation
- Automated notifications
- Protected technical sheets
- Trigger-based automation
- Configurable notification intervals
- Production-safe architecture
---
## `sprint-capacity-planner`
Lightweight sprint planning and workload tracking system.
The spreadsheet acts as the primary UI layer while Apps Script provides backend automation.

### Main features

- Automatic developer assignment
- Workload calculation
- Separator-row architecture
- Structured sprint grouping
- Manual automation controls
- Configurable spreadsheet mapping
---
# Architecture Philosophy
These projects intentionally use Google Sheets as a lightweight operational UI.
Goals:
- reduce manual repetitive work;
- minimize human error;
- provide accessible interfaces for non-technical users;
- avoid unnecessary infrastructure complexity for small teams.
Apps Script is used as a backend automation layer on top of spreadsheet-based workflows.
---
# Security Approach
Public portfolio versions contain:
- anonymized demo data;
- placeholder links;
- no production client information;
- no credentials or secrets.
Production environments should remain private.
