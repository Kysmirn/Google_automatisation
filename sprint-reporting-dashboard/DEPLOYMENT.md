# Deployment Guide

## Requirements

- Google account
- Google Sheets
- Google Apps Script
- Optional: clasp CLI for local development

---

## Create Spreadsheet
1. Create a new Google Spreadsheet.
2. Create sprint sheets using format:

dd.mm-dd.mm
Example:
02.06-09.06
09.06-16.06

---

## Required Columns
Each sprint sheet must contain:
Developer | Task | Priority | Manager | Backlog | Project | Notes | Status
___________________________________________________________________________

## Install Script
1.	Open: 
Extensions → Apps Script
2.	Copy contents of: 
src/statistics_dashboard.js
into Apps Script editor.
3.	Copy contents of: 
src/appsscript.json
into manifest file.

--- 

## Enable Manifest File
If appsscript.json is not visible:
Project Settings →
Enable "Show appsscript.json manifest file in editor"

---

## Save Project
Press:
Ctrl + S

---

## Reload Spreadsheet
Reload the spreadsheet page.
A new menu should appear:
 Sprint Analytics

---

## Generate Report
Run:
Sprint Analytics → Generate Analytics Report
The script will:
•	detect sprint sheets; 
•	calculate task statuses; 
•	generate analytics; 
•	build charts automatically. 

---

## Notes
•	Sprint sheets are detected automatically by name pattern. 
•	Task status is generated by backend logic. 
•	Spreadsheet formulas are not required. 
•	Demo data should be anonymized before publishing screenshots. 

---

## Recommended Workflow
•	Use Google Sheets as operational UI 
•	Keep business logic inside Apps Script 
•	Use GitHub for version control 
•	Use demo spreadsheets for portfolio screenshots
